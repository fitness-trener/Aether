Install in 5 seconds

One pip command, zero third-party runtime deps. The toolchain runs on stdlib-only Python 3.10+.

pip install aether-lang
aether check program.aeth
aether run   program.aeth

Built for fix-loops

Every diagnostic carries a structured extra dict. The SDK's edit() primitive lets an agent transform the AST and re-render in one call.

from aether import sdk
r = sdk.check(source)
for d in r.diagnostics:
    print(d.code, d.extra)

Live in your editor

Working stdio LSP. Wire it into VS Code's generic LSP client and you get live [E0801], [E0701], [E0302] diagnostics with hover.

python3 -m transpiler.aether.lsp
